# Investment vault

This directory contains the repository's authoritative, human-maintained
investment content:

- `portfolio/` records the mandate, factual portfolio state, review schedule,
  evaluation log, and investor decisions.
- `companies/` holds company research, models, structured snapshots, and
  immutable source evidence.
- `templates/` provides starting points for repeatable research artifacts.
- `schemas/` defines contracts for structured records in the vault.

Applications and tools outside this directory may index, render, or validate
the vault, but they must not become a competing source of investment truth.
